# @atlas/agent-tools

This package will host the agent SDK — the nine MCP tools described in [`specs/08-AGENT-ECOSYSTEM-SPEC.md`](../../specs/08-AGENT-ECOSYSTEM-SPEC.md) plus LangChain bindings for the same surface, so any agent runtime can discover events, fetch ticket types, drive the HTTP 402 purchase flow, and read receipts against an ATLAS registry.
